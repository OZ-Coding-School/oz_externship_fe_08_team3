## 막혔던 것

- `overflow-hidden`이 CSS에서 스크롤 컨테이너를 생성하여 `scrollIntoView`가 위젯 자체를 스크롤시키는 문제 → `overflow-clip`으로 해결
- MessageList 부모가 flex 컨테이너가 아니어서 `flex-1`이 동작하지 않는 문제 → `flex flex-col` 추가로 해결
