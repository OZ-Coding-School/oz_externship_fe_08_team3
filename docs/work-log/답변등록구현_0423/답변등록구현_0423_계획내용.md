## API 명세

### 답변 등록 (구글 시트 89번)

- **Method / Endpoint:** `POST /api/v1/qna/questions/{question_id}/answers`
- **Request Header:** `Authorization: Bearer {token}`, `Content-Type: application/json`
- **Request Body:**
  ```json
  { "content": "str", "img_urls": ["str"] }
  ```
- **Response 201:**
  ```json
  { "answer_id": bigint, "question_id": bigint, "author_id": bigint, "created_at": "str" }
  ```
- **Error:** 400, 401, 403, 404

### 이미지 업로드 Presigned URL (구글 시트 125번)

- **Method / Endpoint:** `PUT /api/v1/qna/answers/presigned-url`
- **Request Body:** `{ "file_name": "str" }`
- **Response 200:**
  ```json
  { "presigned_url": "str", "img_url": "str", "key": "str" }
  ```
- **Error 400:** `{ "error_detail": "str" }`
